# Output

Build a website that controls real-world hardware and we'll send you parts like RPi Zero 2Ws, LED matrices, stickers & more to make it happen!

**Output** is a [Hack Club](https://hackclub.com) You Ship We Ship (YSWS) program where teens build websites that interact with physical hardware. Once your project is approved, we ship you the parts you need!

## Examples

- **[print.lukasreis.com](https://print.lukasreis.com)** ~ Leave a message on the website → it gets printed to a physical printer!
- **[CompilerFax](https://github.com/lexbailey/compilerfax)** ~ Fax your C code to a Raspberry Pi → it compiles it and faxes back the result!
- **[inkday](https://github.com/jflessau/inkday)** ~ Upload images via a website calendar → they show up on a physical e-ink display each day.
